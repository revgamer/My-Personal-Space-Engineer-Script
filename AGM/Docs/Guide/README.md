# AGM v1.3 Guides

Read these in order:

1. `AGM_Setup_Step_By_Step.md`
2. `AGM_v1.3_Custom_Data.md`
3. `AGM_v1.3_LCD_Dashboards.md`

Use `AGM_Setup_Step_By_Step.md` first. It explains what to do before adding screens, how to set up the programmable block, and how to configure base-only block groups.

Use `AGM_v1.3_Custom_Data.md` when editing PB Custom Data.

Use `AGM_v1.3_LCD_Dashboards.md` when adding or fixing LCD pages.
