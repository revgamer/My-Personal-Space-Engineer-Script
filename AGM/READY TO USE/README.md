# Ready To Use

## Install AutoGrid Manager

1. Open `AGM.cs` in this folder.
2. Copy the complete script into a Space Engineers programmable block.
3. Press **Check Code**.
4. Press **Remember & Exit**.
5. Run the programmable block once.
6. Wait for the AGM boot bar to reach 100 percent.
7. Follow `AGM-Full-Guide.md` to tag cargo, machines, LCDs, alerts, and docks.

## Files

- `AGM.cs` is generated by the local Ingame Script Merge Tool and is the file
  intended for the programmable block.
- `AGM-Full-Guide.md` contains the complete setup with examples for every
  section.

Do not edit `AGM.cs` as the development source. Make code changes in
`../Developer/AutoGrid-Manager-v3.0.cs`, then regenerate the minified copy.
