# AutoGrid Manager v3.0

AutoGrid Manager is a Space Engineers programmable-block inventory, logistics,
monitoring, control, and LCD system.

## Start Here

Open:

```text
READY TO USE/
```

That folder contains:

- `AGM.cs`: the minified paste-ready programmable-block script.
- `AGM-Full-Guide.md`: complete step-by-step setup, examples, tags, screens,
  configuration, commands, and troubleshooting.
- `README.md`: short installation checklist.

## Folder Layout

```text
AGM/
|-- READY TO USE/   Paste-ready script and complete user guide
|-- Developer/      Editable source, minification project, and recovery archive
|-- Guide/          Canonical full documentation and guide index
|-- Reference/      GOAT, IIM, block TypeId, and item TypeId references
|-- README.md       This folder map
`-- .gitignore      Build-output exclusions
```

Autocrafting and automatic disassembly are intentionally excluded from v3.0.
Assemblers are monitored and their completed output is still sorted.
