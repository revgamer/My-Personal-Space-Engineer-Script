HORIZON SENTINEL CUSTOM DATA GUIDE
==================================

This file is the setup guide for Horizon Sentinel.

There are four different places tags/settings can go:

1. Programmable Block Custom Data
   - Main settings and block group names.

2. LCD / cockpit / control seat block name
   - Add {HSLCD} or {HSCLCD} to the block name so Horizon Sentinel can find the screen.

3. LCD / cockpit / control seat Custom Data
   - Add one or more [HS:Page] tags so Horizon Sentinel knows what to draw.

4. Cargo blocks / air vents Custom Data
   - Optional block tags for exact cargo and pressurization selection.


QUICK START
===========

1. Put this in the Programmable Block Custom Data:

[Horizon Sentinel]
Auto Reactor Charging=false
Auto Hydrogen Balancing=false
Auto Ammo Balancing=false

Group Batteries=HS Batteries
Group Reactors=HS Reactors
Group Solar Panels=HS Solar Panels
Group Hydrogen Engines=HS Hydrogen Engines

Group Hydrogen Tanks=HS Hydrogen Tanks
Group Oxygen Tanks=HS Oxygen Tanks
Group Jump Drives=HS Jump Drives
Group Turrets=HS Turrets
Group Cargo=HS Cargo
Group Air Vents=HS Air Vents

Group Thrusters=HS Thrusters
Group Lifting Thrusters=HS Lifting Thrusters
Group Cruising Thrusters=HS Cruising Thrusters
Group Braking Thrusters=HS Braking Thrusters
Group Gyros=HS Gyros

2. Name normal LCDs/cockpits/control seats with:

{HSLCD}

3. Name corner LCDs/small LCDs with:

{HSCLCD}

4. Put a page tag in the screen Custom Data, for example:

[HS:Cargo]
Surface=0
Layout=Auto

5. After changing code or Custom Data, press Check Code then OK. Horizon Sentinel refreshes itself.


ALL SCREEN NAME TAGS
====================

These go in the block name, not Custom Data.

{HSLCD}
Use for normal LCDs, wide LCDs, control seats, cockpits, and flight seats.

{HSCLCD}
Use for corner LCDs or small status screens. This forces compact display mode.

Examples:

Bridge Cargo LCD {HSLCD}
Pilot Seat {HSLCD}
Power Corner LCD {HSCLCD}

SCREEN OPTIONS
==============

Put these under a screen page tag.

Surface=0

Surface chooses which screen surface to use.

Normal LCD:
Surface=0

Control seats / cockpits:
Surface=0 usually first screen
Surface=1 usually second screen
Surface=2 usually third screen


Layout=Auto

Layout controls screen style.

Layout=Auto
Horizon Sentinel chooses from screen size.

Layout=Full
Forces the full normal page.

Layout=Bar
Forces a compact horizontal bar page.

Layout=Stack
Forces a compact vertical/portrait page.

Important:
If the whole LCD block is physically rotated sideways in the ship, the PB cannot know your camera/viewing direction.
Use Layout=Bar or Layout=Stack to improve the page shape, but rotate the LCD block in build mode if text itself is sideways.


PROGRAMMABLE BLOCK CUSTOM DATA DETAILS
======================================

Automation settings:

Auto Reactor Charging=false
If true later automation can turn reactors on/off for battery charging.

Auto Hydrogen Balancing=false
If true later automation can balance hydrogen tank stockpile states.

Auto Ammo Balancing=false
If true later automation can balance turret ammunition.


Group settings:

Group Batteries=HS Batteries
Battery group. If missing, Horizon Sentinel scans same-grid batteries.

Group Reactors=HS Reactors
Reactor group. Optional.

Group Solar Panels=HS Solar Panels
Solar panel group. Optional.

Group Hydrogen Engines=HS Hydrogen Engines
Hydrogen engine group. Optional.

Group Hydrogen Tanks=HS Hydrogen Tanks
Hydrogen tank group. If missing, Horizon Sentinel tries to identify hydrogen tanks by block subtype/name.

Group Oxygen Tanks=HS Oxygen Tanks
Oxygen tank group. If missing, Horizon Sentinel tries to identify oxygen tanks by block subtype/name.

Group Jump Drives=HS Jump Drives
Jump drive group. Recommended if your ship has more than one jump drive.

Group Turrets=HS Turrets
Turret group. If missing, Horizon Sentinel scans same-grid turrets.

Group Cargo=HS Cargo
Cargo/inventory group. Optional if using [HS:CargoBlock] tags.

Group Air Vents=HS Air Vents
Air vent group. Optional if using [HS:PressureVent] tags.

Group Thrusters=HS Thrusters
All thrusters group.

Group Lifting Thrusters=HS Lifting Thrusters
Thrusters that hold the ship up against gravity. This is important for planet entry safety.

Group Cruising Thrusters=HS Cruising Thrusters
Forward/cruising thrusters.

Group Braking Thrusters=HS Braking Thrusters
Reverse/braking thrusters.

Group Gyros=HS Gyros
Gyroscope group.


BLOCK CUSTOM DATA TAGS
======================

These go into the actual cargo blocks or air vents, not LCDs.


Cargo block tag:

[HS:CargoBlock]

Use on:
- cargo containers
- connectors
- sorters
- cockpits
- any inventory block you want counted as ship cargo

Do not use on:
- inventories you do not care about
- production blocks you do not want counted
- temporary transfer connectors

Behavior:
If at least one inventory block has [HS:CargoBlock], cargo status uses only tagged inventory blocks.
If no [HS:CargoBlock] tags exist, Horizon Sentinel uses Group Cargo.
If Group Cargo is missing too, it falls back to same-grid inventory blocks.


Pressure vent tag:

[HS:PressureVent]

Use on:
- vents that should remain pressurized
- living areas
- bridge/cockpit areas
- sealed crew spaces

Do not use on:
- depressurization vents
- hangar drain vents
- airlock drain vents
- vents intentionally set to Depressurize

Behavior:
If at least one air vent has [HS:PressureVent], pressure status uses only tagged vents.
If no [HS:PressureVent] tags exist, Horizon Sentinel uses Group Air Vents.
If Group Air Vents is missing too, it falls back to same-grid vents.


SCREEN PAGE GUIDE
=================


1. SPLASH / LOGO
----------------

Tag:

[HS:Splash]
Surface=0
Layout=Full

Best screen:
- PB screen
- normal LCD

Shows:
- Horizon Sentinel logo
- ocean/horizon background
- shield and crosshair pulse

Data needed:
- none


2. PILOT STATUS
---------------

Tag:

[HS:Pilot]
Surface=0
Layout=Full

Best screen:
- control seat
- cockpit
- normal LCD

Shows:
- battery charge
- hydrogen fuel
- oxygen level
- gravity
- ship mass
- jump charge summary
- master ship safety state

Data sources:
- Group Batteries
- Group Hydrogen Tanks
- Group Oxygen Tanks
- Group Jump Drives
- main cockpit/ship controller for mass and gravity

If something is missing:
- no batteries means power percent can be 0
- no tanks means fuel/oxygen can be 0
- no ship controller means mass/gravity may not be available


3. JUMP DRIVES

Corner/small version:

[HS:Jump]

Shows:

- half gauge on full page
- online count
- damaged count
- state: READY, CHARGING, OFFLINE, DAMAGED, NO DRIVES

Data source:
- Group Jump Drives

Recommended:

4. CARGO STATUS
---------------

Tag:

[HS:Cargo]
Surface=0
Layout=Full

Compact/wall version:

[HS:Cargo]
Surface=0
Layout=Bar

Shows:
- cargo fill percent
- cargo mass
- cargo block count
- cargo density warning
- state: OK, HEAVY, FULL

Data source priority:
1. inventory blocks tagged [HS:CargoBlock]
2. Group Cargo
3. same-grid inventory blocks

Recommended if you do not want all inventory counted:
Put [HS:CargoBlock] in only the cargo blocks you want monitored.


5. PLANET ENTRY / DESCENT
-------------------------

Tag:

[HS:Interplanetary]
Surface=0
Layout=Full

Shows:
- natural gravity percent
- hydrogen fuel
- lift capacity
- cargo load
- ship mass
- vertical speed
- lift thruster online count
- safety state

Safety states:
- SPACE
- SAFE
- CAUTION
- DO NOT ENTER
- FALLING
- NO LIFT
- LIFT OFF
- LIFT PARTIAL
- NO HOVER
- LOW LIFT

Data sources:
- Group Hydrogen Tanks
- Group Lifting Thrusters
- Group Cargo or [HS:CargoBlock]
- ship controller for mass, gravity, vertical speed

Important:
Group Lifting Thrusters must contain the thrusters that keep the ship up in planetary gravity.
If you turn off lifting thrusters in gravity, this page should stop saying SAFE.

8. PRESSURE
-----------

Tag:

[HS:Pressure]
Surface=0
Layout=Bar

Pressure is intended for corner LCDs and compact status screens.

Shows:
- average pressure percent
- pressurized vent count
- state: PRESSURIZED, PARTIAL, LEAK / NO AIR, NO VENTS

Data source priority:
1. vents tagged [HS:PressureVent]
2. Group Air Vents
3. same-grid air vents

Do not tag depressurization vents.

TROUBLESHOOTING
===============

Screen does not change:
- Block name must contain {HSLCD} or {HSCLCD}.
- Screen Custom Data must contain a valid [HS:Page] tag.
- Run Check Code then OK after editing the script.

Wrong cockpit screen:
- Change Surface=0 to Surface=1 or Surface=2.

LCD looks squeezed:
- Try Layout=Bar.
- Try Layout=Stack for tall screens.
- Try Layout=Full for normal LCDs.

Text is physically sideways:
- Rotate the LCD block in build mode.
- Use the game's LCD rotation setting if that block has one.
- PB script can choose layout, but it cannot know your viewing angle.

Cargo includes too many blocks:
- Put [HS:CargoBlock] only on the inventory blocks you want counted.

Pressure includes depressurization vents:
- Put [HS:PressureVent] only on vents that should be pressurized.

Planet entry says NO LIFT / LIFT OFF:
- Check Group Lifting Thrusters.
- Make sure lifting thrusters are online and working.

Planet entry says NO HOVER:
- Ship is too heavy for working lift thrust in current gravity.
- Add lift thrust, reduce cargo, or avoid planet entry.
